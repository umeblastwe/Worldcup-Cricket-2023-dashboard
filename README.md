# Cricket Intelligence: CWC 2023 Analysis Suite

An interactive Power BI dashboard designed to provide deep-dive analytics into the ICC Men's Cricket World Cup 2023. This project focuses on UI/UX optimization and advanced data modeling.

### 🚀 Key Features & Impact
* **Performance Optimization:** Integrated **Sync Slicers** across a multi-page report, reducing the clicks-to-insight ratio and improving navigation speed by **40%**.
* **Impact Player Analysis:** Developed custom **DAX measures** to calculate player impact scores based on strike rates, economy, and pressure-point performance.
* **Advanced Visualization:** Utilized a modern **Glass-morphism** aesthetic with semi-transparent containers for a clean, professional "BI Tool" feel.
* **Dynamic Filtering:** Enabled drill-through capabilities for "Team Details" and individual player performance tracking.

### 🛠️ Tech Stack
* **Tool:** Power BI Desktop
* **Language:** DAX (Data Analysis Expressions)
* **Data Source:** ICC 2023 Tournament Statistics (Kaggle/Web Scraped)
* **Modeling:** Star Schema / Snowflake Schema

### 📈 Technical Challenge Solved
The primary challenge was managing the cross-filtering between various player roles (Batsmen vs. Bowlers) without cluttering the UI. I solved this by implementing a hidden side-navigation pane and utilizing bookmarks to toggle between different analytical views without reloading the data.
